# BaseCamp Linux — Community Plugins

Optional plugins for [BaseCamp Linux](https://github.com/Ramisotti/mountain-time-sync).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [Philips Hue](hue_control/) | Control Hue lights & scenes from DisplayPad buttons |

## Installation

1. Open BaseCamp Linux
2. Go to the **Plugins** panel
3. Click **Install Plugin**
4. Paste the plugin URL or browse to a downloaded plugin folder

### Manual Installation

Download or clone the plugin folder into:
```
~/.config/mountain-time-sync/plugins/
```
Then restart BaseCamp Linux.
